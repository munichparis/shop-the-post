This plugin adds shop the post widgets from different affiliate networks to the homepage.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `shop-the-post.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
